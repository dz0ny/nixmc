Capture my current non-default macOS settings, Homebrew apps, shell tools, and launchd items into the flake so a replacement Mac feels identical.

This is an inventory and migration workflow: the existing machine's unmanaged
state must be discovered before it can be expressed declaratively, so no Nix
option query applies.
